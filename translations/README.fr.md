# Awesome README Resources

Une liste sélectionnée de ressources pour améliorer les READMEs de profils
GitHub, les READMEs de projets, les badges, les cartes de statistiques, les
diagrammes, les captures d'écran et les petits détails qui rendent un dépôt plus
clair.

Rends ton README utile d'abord, puis agréable à regarder.

## Démarrage Rapide

- 📘 [About READMEs](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-readmes) - La documentation officielle de GitHub sur le rôle d'un README.
- ✍️ [Basic writing and formatting syntax](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) - Syntaxe Markdown prise en charge par GitHub.
- 🛠️ [Make a README](https://www.makeareadme.com/) - Guide simple pour écrire un README de projet.
- 🧱 [readme.so](https://readme.so/) - Éditeur visuel de README avec des sections prêtes à l'emploi.

## README de Profil

- 🖼️ [Awesome GitHub Profile README](https://github.com/abhisheknaiidu/awesome-github-profile-readme) - Galerie de profils GitHub inspirants.
- 🧰 [GitHub Profile README Generator](https://github.com/rahuldkjain/github-profile-readme-generator) - Générateur de sections de profil, liens et stacks techniques.
- 🪧 [GitHub Profile Header Generator](https://leviarista.github.io/github-profile-header-generator/) - Crée des images d'en-tête pour profils et projets.

## Finitions Visuelles

- 🛡️ [Shields.io](https://shields.io/) - Badges personnalisables pour builds, versions, téléchargements et labels.
- 🎨 [Simple Icons](https://simpleicons.org/) - Icônes de marques et couleurs officielles.
- 📊 [GitHub Stats Extended](https://github.com/stats-organization/github-stats-extended) - Cartes de statistiques pour profils GitHub.
- ⌨️ [Readme Typing SVG](https://github.com/DenverCoder1/readme-typing-svg) - Effet de texte animé pour titres de profil.
- 🌊 [Capsule Render](https://github.com/kyechan99/capsule-render) - En-têtes et pieds de page SVG pour READMEs.

## Contribuer

Les nouvelles contributions sont les bienvenues. Ajouter une ressource utile
peut prendre cinq minutes ou moins:

1. Trouve une ressource qui aide à améliorer un README.
2. Ajoute-la dans la section la plus adaptée.
3. Ouvre une pull request avec une courte explication.
