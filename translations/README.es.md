# Awesome README Resources

Una lista curada de recursos para mejorar READMEs de perfiles de GitHub,
READMEs de proyectos, badges, tarjetas de estadísticas, diagramas, capturas de
pantalla y otros detalles que hacen que un repositorio sea más claro.

Haz que tu README sea útil primero y bonito después.

## Inicio Rápido

- 📘 [About READMEs](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-readmes) - Guía oficial de GitHub sobre qué debe comunicar un README.
- ✍️ [Basic writing and formatting syntax](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) - Sintaxis Markdown compatible con GitHub.
- 🛠️ [Make a README](https://www.makeareadme.com/) - Guía práctica para escribir READMEs de proyectos.
- 🧱 [readme.so](https://readme.so/) - Editor visual de README con secciones listas para usar.

## README de Perfil

- 🖼️ [Awesome GitHub Profile README](https://github.com/abhisheknaiidu/awesome-github-profile-readme) - Galería de perfiles de GitHub creativos.
- 🧰 [GitHub Profile README Generator](https://github.com/rahuldkjain/github-profile-readme-generator) - Generador para secciones de perfil, enlaces y stacks técnicos.
- 🪧 [GitHub Profile Header Generator](https://leviarista.github.io/github-profile-header-generator/) - Crea imágenes de cabecera para perfiles y proyectos.

## Toques Visuales

- 🛡️ [Shields.io](https://shields.io/) - Badges flexibles para builds, versiones, descargas y etiquetas personalizadas.
- 🎨 [Simple Icons](https://simpleicons.org/) - Iconos de marcas y colores oficiales.
- 📊 [GitHub Stats Extended](https://github.com/stats-organization/github-stats-extended) - Tarjetas de estadísticas para perfiles de GitHub.
- ⌨️ [Readme Typing SVG](https://github.com/DenverCoder1/readme-typing-svg) - Efecto de escritura animada para titulares de perfil.
- 🌊 [Capsule Render](https://github.com/kyechan99/capsule-render) - Cabeceras y pies SVG para READMEs.
