# Awesome README Resources

GitHub プロフィール README、プロジェクト README、バッジ、統計カード、図、
スクリーンショットなどを見やすくするためのリソース集です。

まず役に立つ README にして、そのあと見た目を整えましょう。

## クイックスタート

- 📘 [About READMEs](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-readmes) - README が伝えるべき内容についての GitHub 公式ガイド。
- ✍️ [Basic writing and formatting syntax](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) - GitHub で使える Markdown 記法。
- 🛠️ [Make a README](https://www.makeareadme.com/) - プロジェクト README の実用的な書き方ガイド。
- 🧱 [readme.so](https://readme.so/) - セクションを組み合わせて README を作れるビジュアルエディタ。

## プロフィール README

- 🖼️ [Awesome GitHub Profile README](https://github.com/abhisheknaiidu/awesome-github-profile-readme) - GitHub プロフィール README の参考例。
- 🧰 [GitHub Profile README Generator](https://github.com/rahuldkjain/github-profile-readme-generator) - プロフィール項目、リンク、技術スタックを作れるジェネレーター。
- 🪧 [GitHub Profile Header Generator](https://leviarista.github.io/github-profile-header-generator/) - プロフィールやプロジェクト用のヘッダー画像を作成。

## 見た目を整える

- 🛡️ [Shields.io](https://shields.io/) - ビルド、バージョン、ダウンロード数などのカスタムバッジ。
- 🎨 [Simple Icons](https://simpleicons.org/) - ブランドアイコンと公式カラー。
- 📊 [GitHub Stats Extended](https://github.com/stats-organization/github-stats-extended) - GitHub プロフィール用の統計カード。
- ⌨️ [Readme Typing SVG](https://github.com/DenverCoder1/readme-typing-svg) - タイピング風のアニメーション SVG。
- 🌊 [Capsule Render](https://github.com/kyechan99/capsule-render) - README 用の SVG ヘッダーやフッター。

## コントリビュート

初めての方の貢献も歓迎します。便利なリソースを 1 つ追加するだけなら、
5 分以内でできることもあります。

1. README を改善するのに役立つリソースを見つける。
2. いちばん合うセクションに追加する。
3. 短い説明を添えて pull request を開く。
