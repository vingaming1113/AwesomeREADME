# Agent Instructions

This repository is an awesome list for README resources. Keep changes focused on
helping people improve GitHub profile READMEs, project READMEs, documentation
visuals, badges, automation, and community files.

## Editing Guidelines

- Keep entries short, practical, and easy to scan.
- Add resources to the most specific existing section before creating a new one.
- Use the existing format: `- emoji [Resource Name](https://example.com/) - Short description.`
- Prefer stable, official, or actively maintained resources.
- Do not add affiliate-heavy, spammy, abandoned, or low-signal links.
- Keep the README useful first and decorative second.
- Run `git diff --check` before committing Markdown changes.

## Commit Guidelines

- Keep unrelated changes in separate commits.
- Use concise commit messages in imperative mood.
- Do not rewrite history or revert user changes unless explicitly asked.
