# Claude Instructions

Follow the repository guidance in [AGENTS.md](AGENTS.md).

## Project Context

Awesome README Resources is a curated Markdown list. The primary job is to keep
the README accurate, welcoming, and easy to browse.

## Preferred Workflow

- Check existing sections before adding new ones.
- Keep descriptions concise and beginner-friendly.
- Preserve the visual style in the README, including resource emojis.
- If adding a generated card, badge, or image service, make sure it has useful
  alt text and a stable target link.
- For contribution-related edits, keep the tone welcoming to first-time
  contributors.
